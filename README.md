# VimSE

VimSE is a general purpose 'runtime library' for Vim.

## How to Install

Using ``vim-plug``, utilizing VimSE is as easy as adding this to your ``.vimrc`` file:
```vimscript
call plug#begin('path-to-your-plugin-folder')
Plug 'MarcelSimader/VimSE'
call plug#end()
```
